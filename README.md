# Actividad 1.3 — Continuous Integration (CI) con GitHub Actions

Este repositorio contiene un proyecto **HTML + CSS + JS** muy simple para demostrar CI.

## ¿Qué se hizo?
- Se creó un workflow de **GitHub Actions** en: `.github/workflows/ci.yml`
- El workflow corre en cada **push** o **pull request** y valida que existan:
  - `index.html`
  - `styles.css`
  - `app.js`

## Cómo ver la página
1. Descarga el repositorio (ZIP)
2. Abre `index.html` en el navegador

## Evidencias (pantallazos)
- Tab **Actions** (run en verde ✅)
- **Logs** del run
- `ci.yml` abierto
- Check ✅ en commit o PR
